What Do We Do

Ne Yapsam is a useful project to evaluate your foods.By using this you can reap a profit and cook delicious foods.

Our team's goal is that making a useful application to help you on preparing dishes at home in this project.

Application will follow your food stock at home and will give you some suggestions at 'What to make for meal?

About Us

We are all last grade students in Eskisehir Technical University.

This Project is our Software Engineering Course Project.

User Requirements

- User should register Ne Yapsam to take advantages of application.

- User should enter his/her ingredients to get a food list which he/she can cook by using those ingredients.

System Requirements

- System should provide proper food list to say users to make them know what kind of foods they can prepare.

- System should provide alternative food list if user suplies missing ingredients.

- System should provide registration capability to the users.

Functional Requirements

- Registered users will be able to select ingredients.

- Registered users will be able to search foods which they can cook.

Non-functional Requirements

1-Product Requirements

- Ne Yapsam should be able to use on all platforms.

2-Organizational Requirements

Users should make a gain after using Ne Yapsam Application.

3-External Requirements

- The system has to make guarantee users privacy.

Usability requirements

- Ne Yapsam should be easy to use.

- Ne Yapsam should has rapid user response capability.
